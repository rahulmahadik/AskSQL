# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

Be respectful and constructive. Harassment, discrimination, or abusive
behavior is not tolerated. Report concerns privately to the maintainers via the
repository's **Security -> Report a vulnerability** channel; reports are handled
confidentially.
