# Connect a database

AskSQL talks to **PostgreSQL**, **MySQL / MariaDB** and **SQLite**, local or cloud.

- Connect with a host and port plus an SSL/TLS mode, or paste a full connection string.
- Passwords and connection strings are stored in your **OS keychain**, never in settings.
- SQLite only needs a path to the `.db` file.

Run **Add Database Connection** to add your first one. It then appears in the **Databases** view, where you can browse tables, views and columns.
