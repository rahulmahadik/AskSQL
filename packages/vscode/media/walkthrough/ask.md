# Ask a question

Open the **AskSQL** panel from the activity bar and ask in plain language, for example:

> *Which customers placed the most orders last month?*

Every answer shows the exact **SQL it ran**, with a short explanation.

- **Read-only by design:** a deterministic guard — not the prompt — decides what runs. Writes, DDL and stacked statements are refused before the database sees them.
- Turn on **Require approval** in settings to review each query before it runs.
- Ask **"describe the orders table"** to get structure straight from the schema, with no query and no model call.
